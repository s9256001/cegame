Baccarat3D 控牌
=========================
0. **控牌基本原則**
	- 6 張牌任意排列
0. **不出閒對、莊對**
	~~~
	閒:   44.50 %
	莊:   45.62 %
	和:    9.88 %
	閒對:  0.01 %
	莊對:  0.01 %
	~~~
	- 無法達成的情況，牌值 map 為
		- [6]、[5 1]
0. **不出和**
	~~~
	閒:   49.63 %
	莊:   50.29 %
	和:    0.09 %
	閒對:  7.75 %
	莊對:  7.80 %
	~~~
	- 無法達成的情況，牌值 map 為
		- [6]、[5 1]
0. **指定閒贏**
	~~~
	閒:   99.83 %
	莊:    0.01 %
	和:    0.16 %
	閒對:  7.10 %
	莊對:  8.94 %
	~~~
	- 無法達成的情況，牌百家樂點數 map 為
		- [6]：和
		- [4 2]：[H05 C05 C05 C05 C12 H13]、[S10 H12 D05 C11 C13 D05]；可能和或莊贏
		- [5 1]：[H03 D03 D03 S03 D03 H13]；和
0. **指定莊贏**
	~~~
	閒:    0.00 %
	莊:   99.92 %
	和:    0.08 %
	閒對:  8.45 %
	莊對:  6.93 %
	~~~
	- 無法達成的情況，牌百家樂點數 map 為
		- [6]：和
		- [4 2]：[C02 S03 D03 D02 H02 H02]；可能和或閒贏
		- [5 1]：[H03 D03 D03 S03 D03 H13]；和
0. **指定閒贏，不要閒對與莊對**
	~~~
	閒:   99.73 %
	莊:    0.05 %
	和:    0.22 %
	閒對:  0.10 %
	莊對:  0.10 %
	~~~
	- 保證可達成的情況，牌百家樂點數 map 為 [2 2 2]
0. **指定莊贏，不要閒對與莊對**
	~~~
    閒:    0.06 %
    莊:   99.78 %
    和:    0.16 %
    閒對:  0.09 %
    莊對:  0.09 %
	~~~
	- 保證可達成的情況，牌百家樂點數 map 為 [2 2 2]
